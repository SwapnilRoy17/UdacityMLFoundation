# UdacityMLFoundation
Analysis on Bike Share Data of three US cities (New York, Chicago, Washington)

Statistics to be found:

1)Most popular hour, count
2)Total duration, count, average duration
3)Popular station 
	Start station, count
	End station, count
4)Popular trip
5)User type
6)Gender
